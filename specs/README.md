# specs/

Specs are split into two categories:

| Category | Folder | Who fills it |
|---|---|---|
| **User specs** | `user/` | You — product knowledge only you have |
| **AI specs** | `ai/` | Claude Code — derived from your user specs |

## user/

Fill every file here before running Claude Code. These capture what only you know about your product — the vision, the rules, the personas, the journeys.

## ai/

Do not edit these manually. Claude Code fills them automatically in Phase 2 by deriving technical architecture from your user specs. Review them in Phase 3 before generating docs.
